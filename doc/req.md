# ｽﾀｯｸﾁｬﾝ抽選会アプリ 要件定義

## 概要

誕生会で使用する抽選会の乱数生成アプリ。Single Page Application (SPA) の Web アプリとして実装する。

## 機能要件

### 抽選機能

- `MIN_NUM` から `MAX_NUM` までの整数（`TOTAL` 個）をランダムに選出する
- 内部ではシンプルに乱数を生成する
- 同じ番号は複数回選ばれてはいけない（重複なし）
- 全番号が出尽くした場合はその旨を通知する

### ルーレット演出

- 生成された乱数を 3 個のルーレットで表示する
  - 100の位・10の位・1の位それぞれにルーレットを配置
- 各桁のルーレットが回転し、順に停止する演出を行う（100の位 → 10の位 → 1の位）
- 実際には裏側で事前に生成された乱数に基づいて停止位置が決まる
- 演出時間は全体で約 6 秒（各桁約 2 秒）

### キャラクター表示

- `pic/image.png`（ｽﾀｯｸﾁｬﾝ）のイラストを画面に表示する
- キャラクターが吹き出しで喋る形式で番号を発表する
- 画像の白背景はそのまま使用（加工不要）
- 吹き出しメッセージ:
  - 待機時: 「ボタンを押してね！」
  - 抽選中: 「次の番号は…」
  - 結果発表: 「XX番！おめでとう！」等（ランダムでバリエーション）
  - 全番号出尽くし時: 「全部出ちゃった！」

### 抽選履歴

- 過去の当選番号を画面下部に小さく表示して残しておく
- 最新の番号はハイライト表示
- 残り番号数を表示する

### データ永続化

- 抽選結果を localStorage に保存し、ページリロード時にも保持する
- リセットボタンを設け、確認ダイアログ付きで全データを初期化できる

## UI / UX 要件

### 操作方法

- 画面上の「抽選！」ボタンをクリックして抽選を開始する
- 抽選中はボタンを無効化し、連打を防止する

### タイトル

- 画面上部にタイトルを表示する
- タイトル文言: 「ｽﾀｯｸﾁｬﾝ誕生会2026抽選会！！」
- サブタイトル: 「Powered by M5Stack社」

### レイアウト / 表示デバイス

- PC（プロジェクター投影）を主な使用環境とする
- 横長画面に最適化し、大きな文字で遠くからも見えるようにする

### 演出効果

- 効果音あり（Web Audio API で合成）
  - ドラムロール（抽選開始時）
  - カチカチ音（ルーレット回転中）
  - 停止音（各桁停止時）
  - ファンファーレ（結果確定時）
- 紙吹雪アニメーション（結果確定時）
- 画面フラッシュ効果（結果確定時）
- キャラクターのバウンスアニメーション（抽選中）

## 技術要件

- フレームワーク不使用の単一 HTML ファイルで実装（SPA）
- CSS / JavaScript はすべてインライン
- 外部依存なし（CDN 等不使用）
- 効果音は Web Audio API でプログラム的に生成（音声ファイル不要）
